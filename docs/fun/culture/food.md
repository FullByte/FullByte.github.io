# Food Culture

## Japanese Chopstick Faux Pas

| Term       | Reading       | Description                                                      | Notes                            |
| ---------- | ------------- | ---------------------------------------------------------------- | -------------------------------- |
| あげ箸     | Agebashi      | Raising chopsticks above the height of one’s mouth               |                                  |
| 洗い箸     | Araibashi     | Cleaning chopsticks in soup or beverages                         |                                  |
| 合わせ箸   | Awasebashi    | Passing food from one pair of chopsticks to another              | Serious taboo (funeral custom)   |
| 受け箸     | Ukebashi      | Holding out bowl for more while still holding chopsticks         |                                  |
| 移り箸     | Utsuribashi   | Repeatedly eating from the same side dish instead of alternating |                                  |
| うら箸     | Urabashi      | Picking up food and putting it back without eating               |                                  |
| 拝み箸     | Ogamibashi    | Holding chopsticks while expressing thanks/prayer                | Considered rude                  |
| 押し込み箸 | Oshikomibashi | Pushing food deep into one’s mouth with chopsticks               |                                  |
| 落とし箸   | Otoshibashi   | Dropping chopsticks while eating                                 |                                  |
| 返し箸     | Kaeshibashi   | Reversing chopsticks to serve food                               |                                  |
| かき箸     | Kakibashi     | Pushing food into mouth or using chopsticks to scratch body      |                                  |
| かみ箸     | Kamibashi     | Biting chopsticks                                                |                                  |
| くわえ箸   | Kuwaebashi    | Holding chopsticks in one’s mouth                                |                                  |
| こじ箸     | Kojibashi     | Digging into food from the bottom of a dish                      |                                  |
| こすり箸   | Kosuribashi   | Rubbing disposable chopsticks together                           |                                  |
| 探り箸     | Saguribashi   | Stirring food to search for something                            |                                  |
| 刺し箸     | Sashibashi    | Stabbing food with chopsticks                                    |                                  |
| 指し箸     | Sashibashi    | Pointing at people or things with chopsticks                     |                                  |
| じか箸     | Jikabashi     | Using personal chopsticks to take from shared dishes             |                                  |
| 透かし箸   | Sukashibashi  | Eating fish by poking between bones instead of removing them     |                                  |
| せせり箸   | Seseribashi   | Repeatedly poking food                                           |                                  |
| そろえ箸   | Soroebashi    | Tapping chopsticks to align them                                 |                                  |
| たたき箸   | Tatakibashi   | Making noise by tapping chopsticks                               |                                  |
| 立て箸     | Tatebashi     | Sticking chopsticks upright in rice                              | Serious taboo (funeral offering) |
| 違い箸     | Chigaibashi   | Using mismatched chopsticks                                      |                                  |
| ちぎり箸   | Chigiribashi  | Using chopsticks like knife and fork to cut food                 |                                  |
| 調伏箸     | Chōbukubashi  | Placing chopsticks with tips pointing right                      |                                  |
| 涙箸       | Namidabashi   | Letting liquid drip from chopsticks                              |                                  |
| 握り箸     | Nigiribashi   | Gripping chopsticks in a fist                                    |                                  |
| ねぶり箸   | Neburibashi   | Licking chopsticks                                               |                                  |
| 橋箸       | Hashibashi    | Placing chopsticks across a bowl like a bridge                   | Indicates finished (improper)    |
| はね箸     | Hanebashi     | Pushing aside unwanted food                                      |                                  |
| 振り上げ箸 | Furiagebashi  | Raising chopstick tips above hand level                          |                                  |
| 振り箸     | Furibashi     | Shaking food/liquid off chopsticks                               |                                  |
| 惑い箸     | Madoibashi    | Hovering chopsticks indecisively over food                       |                                  |
| 回し箸     | Mawashibashi  | Stirring soup with chopsticks                                    |                                  |
| もぎくわえ | Mogikuwae     | Holding chopsticks sideways in mouth                             |                                  |
| もぎ箸     | Mogibashi     | Eating rice stuck to chopsticks                                  |                                  |
| 持ち箸     | Mochibashi    | Holding dish and chopsticks in one hand                          |                                  |
| 楊枝箸     | Yōjibashi     | Using chopsticks like a toothpick                                |                                  |
| 横箸       | Yokobashi     | Using chopsticks like a spoon                                    |                                  |
| 寄せ箸     | Yosebashi     | Pulling dishes closer using chopsticks                           |                                  |
