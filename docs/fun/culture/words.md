# Unique Words Across Languages

A small collection of interesting words that are hard to translate directly.

## Germany

| Word        | Meaning                                 |
| ----------- | -------------------------------------------- |
| Fernweh     | Longing for faraway places                   |
| Feierabend  | The time after work when you are truly done  |
| Weltschmerz | A deep sadness about the state of the world  |
| Zugzwang    | A situation where every possible move is bad |

## Japan

| Word                     | Meaning                                 |
| ------------------------ | -------------------------------------------- |
| Komorebi (木漏れ日)      | Sunlight filtering through tree leaves       |
| Tsundoku (積ん読)        | Buying books and letting them pile up unread |
| Wabi-sabi (侘寂)         | Beauty in imperfection and impermanence      |
| Ikigai (生き甲斐)        | A reason for being that gives life meaning   |
| Mono no aware (物の哀れ) | Gentle sadness about the passing of things   |
