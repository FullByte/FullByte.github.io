# Text QR Codes

## libqrencode

Install [libqrencode](https://github.com/fukuchi/libqrencode.git):

``` sh
git clone https://github.com/fukuchi/libqrencode.git
cd libqrencode/
./autogen.sh
./configure
make
sudo make install
sudo ldconfig
cmake .
make
```
