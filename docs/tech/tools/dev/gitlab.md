## Clone with HTTPS and Access Token

Create an [access token](https://gitlab.com/-/profile/personal_access_tokens) in gitlab

``` sh
git clone https://<gitlab_username>:<gitlab_token>@gitlab.com/<user>/<repo>.git
```
